NEZplug Version 0.9.5.3

■特徴
本家NEZplugをベースにgccでのコンパイルと
SDL環境による簡易プレイヤーの作成を目的としたバージョンです。

■ライセンス
本家NEZplugに準拠します。

■履歴
0.9.5.3  2014/04/11
MinGW32環境でのコンパイルを可能にした。

0.9.5.2  2014/02/14
NRTDRVで使用する際のスタックポインタのアドレスを修正

0.9.5.1  2013/03/14
NRTDRVの読み込みに対応

0.9.5.0   2011/12/20
本家と混同しないようにバージョンナンバーを変えた。
gccでコンパイルが通るようになっている。
SDLによる簡易プレイヤーを作成した。
